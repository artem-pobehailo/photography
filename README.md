# Photography

Простий статичний сайт-портфоліо для фотографії.

## Опис

Цей репозиторій містить мінімальний фронтенд для демонстрації фотографій: статичну HTML-сторінку, стилі та папку з зображеннями.

## Структура проекту

- [index.html](index.html) — головна сторінка.
- [style.css](style.css) — стилі проекту.
- images/ — папка з фотографіями та іншими медіафайлами.

## Перегляд локально

Найпростіший спосіб — просто відкрити `index.html` у браузері (подвійний клік).
Для коректної роботи деяких фіч або при потребі у CORS-запитах, запустіть локальний сервер, наприклад:

Python 3:

```
python -m http.server 8000
```

Потім відкрийте http://localhost:8000 у браузері.

## Налаштування та залежності

Залежностей немає — це статичний сайт. Шрифти підключені через зовнішні сервіси у HTML (якщо потрібно, додайте локальні файли).

## Додавання контенту

Помістіть зображення у папку `images/` і оновіть розмітку в `index.html`, щоб додати нові картки або секції.
